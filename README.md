# Asphalt 8 OpenCV Hand Gesture Controller

A hand gesture control interface using Python, OpenCV, Mediapipe, and cvzone to control Asphalt 8 gameplay with hand gestures detected via webcam.

## Features

- **Steer Left/Right**: Simulated key presses using hand gestures.
- **Accelerate**: Triggered by hand gestures.
- **Brake**: Hand gesture control.
- **Nitro**: Spacebar simulation using thumb gestures.
- **Pause/Resume**: Visual feedback and simulated game pause when both hands are shown open.

## Requirements

Ensure you have Python 3.8+ installed. You can install the required packages using:

```bash
pip install -r requirements.txt
```

## How to Run

Run the controller script:

```bash
python asphalt.py
```

Press `q` to quit the controller application.
